# TechCon 2024 Conference Website

This is the homepage for the **TechCon 2024** conference website. The project demonstrates the use of semantic HTML, accessibility features, and media embedding.

## 📁 Project Structure
Techno_HTML/
└── techno_website/
├── index.html # Homepage of the website
├── media/ # Folder containing video and captions
│ ├── techcon_promo.mp4
│ └── techno_captions.vtt
└── README.md # Project documentation

## 🔧 Features

- Semantic HTML structure (`<header>`, `<main>`, `<footer>`)
- Navigation menu linking to:
  - About
  - Schedule
  - Register
  - Contact
- Embedded promotional video with captions
- Accessible design
